Simple application to read records from utmp/wtmp/btmp linux files
==================================================================
Tested only on `amd64` (`x86_64`) platform.

## Build
```
$ make
```

## Install
```
$ sudo make install
```

## Unistall
```
$ sudo make uninstall
```

## Show help
```
$ gousers --help
```

## Show users
```
$ gousers
```
